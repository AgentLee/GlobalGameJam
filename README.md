# Heart Home - Global Game Jam 2019

## Description
"What home means to you"

Home is where the heart is. **Heart Home** is a co-op maze where two players must listen carfeully to be able to get to the end of the maze. 

## Controls
### Player One
- WASD
### Player Two
- Arrow keys
